# Face refinement variants

These are candidate replacements for the desktop pet main cartoon image.

- `variant-a-balanced.png`: balanced chibi/anime likeness, keeps cuteness while preserving more facial resemblance.
- `variant-b-anime.png`: cleaner anime mascot direction, sharper hair silhouette and facial lines.
- `variant-c-sticker.png`: round sticker mascot direction, friendlier and softer.
- `variant-d-semi-real.png`: semi-chibi direction, slightly more realistic facial detail.

After one option is selected, replace `../pet-cartoon-idle.png` with the chosen image and use it as the visual reference for follow-up action poses.
